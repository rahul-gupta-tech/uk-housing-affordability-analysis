# uk-housing-affordability-analysis
Combined UK property transaction and earnings data to measure housing affordability across the UK. Developed SQL analyses and Power BI dashboards to identify regions where median house prices are most affordable relative to local incomes.

Data
----
The raw datasets used in this project are publicly available from HM Land Registry Price Paid Data and the Office for National Statistics (ONS). Due to file size considerations, the raw data is not included in this repository.

https://www.gov.uk/government/statistical-data-sets/price-paid-data-downloads
